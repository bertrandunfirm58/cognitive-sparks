# ⚡ Sparks

**AI에게 생각하는 법을 가르치는 13가지 인지 프리미티브 — 단순 계산이 아닌.**

> 목표와 데이터를 주세요. 관찰하고, 추상화하고, 유추하고, 놀고, 통합합니다 — 르네상스 박학자처럼.

[생각의 탄생(Sparks of Genius)](https://en.wikipedia.org/wiki/Sparks_of_Genius) (루트번스타인, 1999) 기반 — 아인슈타인, 피카소, 다빈치, 파인만, 나보코프가 사용한 13가지 사고도구.

---

## 이게 뭔가요?

모든 AI 에이전트 프레임워크는 에이전트에게 **뭘 하라고** 가르칩니다 (도구 호출, 상태 관리).
Sparks는 에이전트에게 **어떻게 생각하라고** 가르칩니다 (관찰, 추상화, 유추, 통합).

```
입력:  목표 + 데이터 (아무 형식)
출력:  핵심 원리 + 근거 + 신뢰도 + 유추
```

### 3계층 아키텍처

```
Layer 0: 신경계
  17가지 생물학적 원리 (역치 발화, 습관화, 측면 억제...)
  상태를 감지한다. 명령하지 않는다. 진짜 신경계처럼.

Layer 1: 13가지 사고도구
  관찰 → 패턴인식 → 패턴형성 → 추상화 →
  유추 → 모형 → 통합 (+ deep 모드에서 6개 추가)
  각 도구가 자율적으로 실행. 중앙 오케스트레이터 없음.

Layer 2: AI 증강
  전략적 망각, 모순 유지, 확률적 사고
  인간은 자연스럽게 하지만 AI는 명시적으로 필요한 것들.
```

## 빠른 시작

```bash
pip install -e .

# 아무 데이터 분석
sparks run --goal "핵심 원리를 찾아라" --data ./my-data/ --depth standard

# 진화 모드 (AutoAgent 방식: 변이 → 테스트 → 유지/롤백)
sparks evolve --goal "핵심 원리를 찾아라" --data ./my-data/ --generations 3
```

### Depth 모드

| 모드 | 도구 | 라운드 | 예상 비용 |
|---|---|---|---|
| `quick` | 핵심 4개 | 1 | ~$0.15 |
| `standard` | 7개 | 2 + 수렴 검증 | ~$1-3 |
| `deep` | 13개 | 최대 5 | ~$5-15 |

## 작동 방식

### Phase 1: 순차 학습 (훈련 바퀴)
```
데이터 → 렌즈 부트스트랩 → 관찰 → 패턴 → 추상화 → 유추 → 모형 → 통합
```
시스템이 집중 렌즈로 관찰하고, 패턴을 찾고, 원리로 추상화하고, 구조적 유추로 테스트하고, 판지 모형을 만듭니다.

### Phase 2: 반복 심화 (피카소 황소)
```
Round 1 결과 → 수면/정리 → 전략적 망각 → 재추출 → 비교
```
피카소가 황소를 그린 것처럼: 사실적으로 시작해서 비본질적 디테일을 단계적으로 제거하여 본질만 남깁니다. 시스템이 발견한 것을 잊고 독립적으로 재발견합니다. 두 라운드 모두에서 살아남은 것 = 진짜 원리.

### Phase 3: 수렴
```
Round 1 원리 vs Round 2 원리 → 구조적 비교 → 수렴?
```
두 라운드가 독립적으로 같은 것을 발견하면, 그건 진짜입니다. 아니면 한 라운드 더.

## 신경계 (17가지 생물학적 원리)

CEO 아님. 오케스트레이터 아님. **신경계** — 진짜 신경과학에서 영감.

| 원리 | 생물학 | 구현 |
|---|---|---|
| 역치 발화 | 뉴런이 누적 → 역치 → 발화 → 리셋 | `SignalPotential`: 신호 누적, 역치에서 발화, 불응기 |
| 흥분/억제 | EPSP + IPSP → 순 신호 | 도구가 각 신호에 +/- 기여 |
| 습관화 | 반복 자극 → 반응 감소 | 반복 패턴 유형 자동 감쇠 |
| 민감화 | 새로운/강한 자극 → 반응 증가 | 이상 감지 시 채널 감도 증가 |
| 반사 | 척수가 뇌를 건너뜀 | 예산 소진 → 즉시 중단 (LLM 없이) |
| 고유감각 | 몸이 자기 위치를 감지 | 시스템이 자기 활동, 비용, 모멘텀 추적 |
| 항상성 | 흥분/억제 균형 | 과활성 도구 자동 억제 |
| 헵 가소성 | "함께 발화하면 함께 연결" | 도구→도구 연결이 성공에 따라 강화 |
| 분산 제어 | 불가사리: 뇌 없이 5개 팔 조율 | 각 도구에 `should_run()` 로컬 규칙 |
| 신호 합산 | 공간적 + 시간적 합산 | 여러 도구가 시간에 걸쳐 하나의 신호에 기여 |
| 예측 코딩 | 뇌가 예측 → 오차만 상위로 전파 | Round 1 = 예측; Round 2는 놀라운 것만 처리 |
| 측면 억제 | 망막이 경쟁으로 경계 선명화 | 도구가 경쟁; 가장 관련 높은 것이 승리, 나머지 억제 |
| 시상 게이팅 | TRN이 감각 입력을 피질로 게이팅 | 관찰이 상태 진입 전 필터링 |
| 아교세포 조율 | 성상세포 = 느린 조율 채널 | (계획) 빠른 이벤트 옆에 느린 "분위기" 채널 |
| 신경조절 | 도파민/NE/ACh/5-HT가 학습 제어 | 보상 신호가 도구 우선순위와 탐색 조절 |
| 수초화 | 자주 쓰는 경로가 빨라짐 | 성공한 도구 체인이 더 좋은 모델 배정 |
| 자율신경 모드 | 교감(전투) ↔ 부교감(휴식) | 이상 → 탐색 모드; 수렴 → 통합 모드 |

## 어블레이션 테스트: 신경계가 도움이 되나?

Claude Opus 4.6으로 테스트, 같은 데이터, 같은 목표:

| 지표 | 신경계 ON | 신경계 OFF |
|---|---|---|
| 신뢰도 | 65% | 82% |
| 커버리지 | 75% | 75% |
| 비용 | **$2.28** | $2.46 |
| 유추 | 4 | 6 |
| 모순 | 0 | 1 |

**정직한 결과**: 이 작은 데이터셋(3파일, 2라운드)에서는 단순 파이프라인이 약간 더 나은 원리를 생성했습니다. 신경계의 장점(피드백 루프, 시냅스 학습, 통합, 비용 절감)은 더 많은 데이터와 라운드에서 누적 효과가 나타납니다.

신경계는 **장기 학습에 대한 투자**이지, 단기 부스트가 아닙니다. 진짜 신경계처럼 — 없어지기 전까지는 작동하는 게 안 느껴집니다.

```
sparks/
├── state.py          # CognitiveState + 17가지 생물학적 신호 타입
├── nervous.py        # 17원리 신경계
├── engine.py         # 파이프라인 오케스트레이션
├── lens.py           # 관찰 렌즈 부트스트래핑
├── configurator.py   # 도메인별 적응형 도구/모델 라우팅
├── persistence.py    # 세션 간 학습 (시냅스 + 지식 베이스)
├── evolution.py      # AutoAgent 방식 진화 루프
├── llm.py            # Claude CLI 또는 Anthropic API 백엔드
├── tools/
│   ├── observe.py    # 도구 #1: 렌즈로 관찰
│   ├── patterns.py   # 도구 #4-5: 패턴 인식 + 형성
│   ├── abstract.py   # 도구 #3: 피카소 황소 추상화
│   ├── analogize.py  # 도구 #6: 구조적 유추
│   ├── model_tool.py # 도구 #10: 판지 모형 테스트
│   └── synthesize.py # 도구 #13: 최종 통합
└── cli.py            # sparks run / evolve / info
```

## 이론적 기반

### 책에서
> "현상은 복잡하다. 법칙은 단순하다. 버릴 게 무엇인지 알아내라." — 파인만

13가지 사고도구(관찰, 형상화, 추상화, 패턴인식, 패턴형성, 유추, 몸으로 생각하기, 감정이입, 차원적 사고, 모형, 놀이, 변형, 통합)는 창의적 사고의 원자적 연산 — 모든 분야에서 과학자, 예술가, 공학자가 동일하게 사용.

### 우리의 가설
> 이 13가지 연산이 진정 보편적이라면, 어떤 데이터에든 체계적으로 적용하면 인간 전문가가 수년간의 경험으로 도달하는 것과 같은 깊이의 이해를 추출할 수 있어야 한다.

### 뭐가 다른가
- **LangGraph/CrewAI**: "도구를 어떻게 조율하나" → 우리: "도구로 어떻게 생각하나"
- **AutoGPT/autoresearch**: "더 빠른 실험" → 우리: "더 깊은 이해"
- **GPT-Researcher/STORM**: "더 나은 검색" → 우리: "더 나은 추상화"
- **뇌 영감 AI**: 연산을 위해 뉴런 모방 → 우리: 조율을 위해 신경계 모방
